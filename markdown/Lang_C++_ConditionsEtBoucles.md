# Conditions et boucles

Condition IF:
```c++
if (<Condition>) <Instructions>
else if (<Condition>) <Instructions>
else <Instructions>

```
Condition SWITCH  

```c++
switch (<Variable>){
	case <Valeur>:
		<Instructions>
	case <Valeur>:
		<Instructions>
	...
}

```
Boucle WHILE:
```c++
while (<Condition>) <Instructions>

```
Boucle FOR
```c++
for (<Declaration et affectation variable>; <Condition sortie>; <Increment>){
<Instructions>
}

```
